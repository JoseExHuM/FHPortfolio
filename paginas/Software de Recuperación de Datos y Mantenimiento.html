Disk Drill

Disk Drill es un programa gratuito para la recuperación de datos que posee una interfaz de usuario limpia y fácil de entender, junto con un eficiente algoritmo de recuperación de datos. Esto proporciona una manera fácil para que los usuarios escaneen la PC en busca de archivos perdidos, dañados o corruptos y los previsualicen antes de someterse al proceso de recuperación.

- Permite recuperar archivos perdidos de particiones que han sido formateadas o no se han podido montar.

- Puede escanear discos duros, particiones que se han borrado o perdido. Los dispositivos de memoria externa como las unidades flash y las tarjetas SD también se pueden ejecutar a través de Disk Drill para la exploración y la recuperación de datos.

- Disk Drill también es capaz de extraer datos de los discos duros que se han dañado.

- Hay diferentes opciones de escaneo y capacidades de escaneo que Disk Drill puede hacer. Las opciones de escaneo rápido y escaneo avanzado se pueden utilizar para buscar archivos perdidos, borrados, corruptos o dañados.

- También es posible clasificar y escanear según el tipo de archivo con Disk Drill.

Disk Drill está disponible en 2 versiones: Mac y Windows. Estas 2 versiones son ligeramente diferentes en la recuperación de datos.
